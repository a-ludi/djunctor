djunctor
=========

Close assembly gaps using long-reads with focus on correctness.


Building
--------

```
git clone https://github.com/a-ludi/djunctor.git
cd djunctor
dub build
```


Developer Notes
---------------

```
# run the program for development
dub run -- ARGS...

# run tests
dub test

# clean up
dub clean
```
